Parte 2 · El Pipeline de Datos

De datos brutos
a output limpio

Paradigma Face-Word Interference (FWI) · Participante P01 · 16 ensayos registrados.

En Excel, procesar esto para 49 participantes requería ~3 horas. Con Python y Claude: 2 minutos.

Datos brutos. 16 ensayos del participante P01 tal como salen del software de registro (E-Prime). Tres condiciones: Congruente, Incongruente y Neutro. TR = Tiempo de Reacción en milisegundos. Precisión: ✓ correcto, ✗ error. Algunos registros ya se ven problemáticos — identifícalos antes de avanzar.
Paso 1: TR = 0. Los ensayos con tiempo de reacción igual a 0 indican ausencia de respuesta (fallo del sistema o sin respuesta del participante). No hay información recuperable. Se eliminan antes de cualquier cálculo. Ensayos eliminados: 1 y 11. Quedan 14 ensayos.
Paso 2: Outliers por criterio absoluto. TR < 150 ms = respuesta anticipatoria (fisiológicamente imposible procesar el estímulo en ese tiempo). TR > 1500 ms = probable distracción o abandono de tarea. Criterio estándar en paradigmas de interferencia. Ensayos eliminados: 5 (muy lento), 7 (anticipatorio), 14 (muy lento). Quedan 11 ensayos válidos.
Paso 3: Resumen por condición. Con los ensayos válidos calculamos TR medio y precisión por condición. Esto es lo que entra al análisis estadístico. El mismo código — exactamente este — corre sobre los 49 participantes de forma automática.

El código Python que hace esto para los 49 participantes
# Pipeline FWI — limpieza automática por participante
resultados = []

for pid in lista_participantes:
    df = pd.read_csv(f"datos/{pid}.csv")

    # Paso 1: eliminar TR inválidos
    df = df[df['TR'] > 0]

    # Paso 2: eliminar outliers absolutos
    df = df[(df['TR'] >= 150) & (df['TR'] <= 1500)]

    # Paso 3: resumen por condición
    resumen = df.groupby('condicion').agg(
        TR_medio  = ('TR', 'mean'),
        precision = ('precision', 'mean'),
        n_validos = ('TR', 'count')
    ).round(2)
    resumen['participante'] = pid
    resultados.append(resumen)

# Un dataframe final con los 49 participantes. Listo para análisis.
df_final = pd.concat(resultados).reset_index()
